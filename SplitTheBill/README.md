# Split The Bill

A simple tip calculator and bill splitter built with Blazor WebAssembly.

## What it does

Enter an expense, pick a tip percentage, and split the total across however many people you like.

## How to run
```bash
cd SplitTheBill
dotnet run
```

Then open your browser at `https://localhost:5001`.

## Notes

- Follows system dark/light theme automatically via `prefers-color-scheme`
- Built with Blazor WebAssembly and Bootstrap 5
- No external dependencies beyond the default Blazor template
